// FSRS ENGINE

The Science of Remembering

Free Spaced Repetition Scheduler: the most advanced open-source scheduling algorithm.

How FSRS Works

ALGORITHM v4.0
STEP 01

Initial Learning

Student encounters new material. FSRS records difficulty signal.

STEP 02

Memory Model

DSR model (Difficulty, Stability, Retrievability) predicts forgetting curve.

STEP 03

Optimal Review

Scheduled at the moment when retrievability drops to target threshold (90%).

STEP 04

Adaptation

Each review updates parameters. Algorithm improves per-student over time.

// FSRS scheduling core (simplified)
function schedule(card, rating) {
  const { difficulty, stability } = card.state;
  const retrievability = Math.exp(
    Math.log(0.9) * elapsedDays / stability
  );
  const newStability = updateStability(
    difficulty, stability, retrievability, rating
  );
  return {
    nextReview: daysUntilTarget(newStability, 0.9),
    state: { difficulty: updateDifficulty(difficulty, rating), stability: newStability }
  };
}
